## 🚀 프로젝트 소개

BEST사이트는 직원들의 업무 스케줄 관리, 기사님의 버스 노선 제공, 내부 공지 및 피드백 수집 기능을 통합하여 버스회사의 운영을 보다 체계적이고 효율적으로 관리할 수 있는 사이트입니다.
